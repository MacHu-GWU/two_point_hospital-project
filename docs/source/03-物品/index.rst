椅子类
------------------------------------------------------------------------------

房间内的椅子可以有助于将员工保持在离出口进的地方, 使得病人到来后可以立即与病人互动开展工作.

房间外的椅子能降低病人在等待期间降低的幸福感, 但是如果在科室附近的椅子不够排队的所有病人坐, 那么病人就可能跑到很远的地方去坐, 还不如不放.
