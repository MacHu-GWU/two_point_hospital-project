Install
------------------------------------------------------------------------------

``two_point_hostpital`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install two_point_hostpital

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade two_point_hostpital
